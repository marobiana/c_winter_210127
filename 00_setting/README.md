### 프로젝트 이름, 저장 위치 설정
- 프로젝트 이름 규칙: 
  - 예제: Dayxx_배우는 내용_ex 예) Day01_variable_ex01
  - 퀴즈: Dayxx_배우는 내용_quiz 예) Day01_variable_quiz01
- 저장 위치: D드라이브 \ 본인 이름 \ workspace
