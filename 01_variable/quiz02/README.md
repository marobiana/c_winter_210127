## 문제) 입력

### 1. 입력 출력
* 나이를 입력 받아서 아래와 같이 출력하세요.
 
> 입력 예시

```
나이를 입력하세요: 24
```

> 출력 예시

```
나이:24
```

### 2. 입력과 더하기 
* 두 수를 입력 받아서 더하고 아래와 같이 출력하세요.


> 입력 예시

```
두 수를 입력하세요 : 3 5
```

> 출력 예시

```
3 + 5 = 8
```

### 3. 입력과 뺴기 
* 두 수를 입력 받아서 빼고 아래와 같이 출력하세요.


> 입력 예시

```
두 수를 입력하세요 : 3 5
```

> 출력 예시

```
3 - 5 = -2;
```

### 4. 구구단
* 단수를 입력 받고 아래와 같이 구구단을 출력하세요.

> 입력 예시

```
단수를 입력하세요 : 5
```

> 출력 예시

```
5 X 1 = 5
5 X 2 = 10
5 X 3 = 15
5 X 4 = 20
5 X 5 = 25
```

### 5. 원 넓이 구하기 
* 지름을 입력 받아서 원의 넓이를 구하여 출력하세요. 


> 입력 예시

```
지름을 입력하세요 : 24
```

> 출력 예시

```
원의 넓이는 452.160004 입니다.
```
