## 문제) 문자열(String)

### 1. 문자열 변수
* 문자열 변수 세개를 이용해서 아래 문장을 출력 하세요.  
`Oh My God`

> 출력 예시

```
Oh My God
```

### 2. 문자열 입력
* id를 입력 받고, 아래와 같이 환영 메시지를 출력하세요.
`Welcome! id`

> 입력 예시

```
ID를 입력하세요 : marobiana
```

> 출력 예시

```
Welcome! marobiana
```

### 3. 문자열 접근
* 아래 문자열에서 A 를 B로 바꿔서 출력 하세요.  
`char gradeStr[] = "My grade is A";`


> 출력 예시

```
My grade is B
```
